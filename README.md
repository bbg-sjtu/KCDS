This is the codebase for the KCDS method.

The folder EF+CNF contains code for training and testing Euclideanizing Flows with Neural ODEs on the LASA 2D dataset.

The folder EF+CNF+3D contains code for training and testing Euclideanizing Flows with Neural ODEs on the LASA 3D dataset.

The folder EF+CNF+FMAug+KCanyon contains code for training and testing KCDS with Neural ODEs and FM enhancements on the LASA 2D dataset.

The folder EF+CNF+FMAug+KCanyon+3D contains code for training and testing KCDS with Neural ODEs and FM enhancements on the LASA 3D dataset.
