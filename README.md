This is the code repository for the KCDS method.

The folder EF+CNF contains code for training and testing Euclideanizing Flows with neural ODEs on a 2D dataset.

The folder EF+CNF+3D contains code for training and testing Euclideanizing Flows with neural ODEs on a 3D dataset.

The folder EF+CNF+FMAug+KCanyon contains code for training and testing KCDS with neural ODEs and FM enhancements on a 2D dataset.

The folder EF+CNF+FMAug+KCanyon+3D contains code for training and testing KCDS with neural ODEs and FM enhancements on a 3D dataset.
